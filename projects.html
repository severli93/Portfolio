Airbnb Boston DataViz

date: December, 2015   instructor: Siqi Zhu   #dataviz #d3.js

Designed and developed a web exploration tool visualizing data about Airbnb rooms in Boston, which consists of geographical distribution and reviews.

This is the final project of Siqi Zhu's Visualization Technology class. The data is a csv file provided by the instructor. I designed and developed the visualization by using D3 javascript and Bootstrap.
